01. 함수 : 선언적 함수

선언적 함수는 자바스크립트에서 함수를 선언할 때 이름을 붙여주는 방식입니다.
이 방식을 사용하면, 함수를 따로 변수에 할당하지 않고도 선언한 이름으로 바로 호출할 수 있어요. 📞

선언적 함수의 특징
* 명시적 이름 부여: 선언적 함수는 함수 선언 시 명시적으로 이름을 부여합니다. 이 이름을 사용하여 함수를 호출할 수 있어요.
* 호이스팅(Hoisting) 혜택: 선언적 함수는 자바스크립트 엔진에 의해 코드의 최상단으로 끌어올려지는 호이스팅의 혜택을 받습니다.
이는 함수를 선언하기 전에도 호출할 수 있게 해줍니다. 

{
    function func() {
        console.log("01. 함수가 실행되었습니다")
    }
} func()

function func() { ... }: 이 부분은 func라는 이름의 함수를 정의하는 부분입니다.
함수 내용은 중괄호 {} 안에 위치하며, 여기서는 "01. 함수가실행되었습니다"라는 문자열을 콘솔에 출력하고 있습니다.

func(): 이 부분은 정의된 func 함수를 호출하는 부분입니다. 이렇게 호출하면 함수 내부의 코드가 실행되고,
"01. 함수가 실행되었습니다"가 콘솔에 출력됩니다.

결과 확인하기
01. 함수가 실행되었습니다

02. 함수 : 익명 함수

익명 함수는 이름이 없는 함수를 가리킵니다.
일반적으로 함수를 정의할 때 함수의 이름을 지정하는데, 익명 함수는 이름을 가지지 않기 때문에 익명 함수라고 부릅니다.
익명 함수는 코드를 보다 간결하게 만들고, 필요한 곳에 간편하게 사용할 수 있는 장점이 있습니다 

{
    const func = function () {
        console.log("02. 함수가 실행되었습니다");
    };
    func();
}

const func = function () { ... };: 이 부분은 익명 함수를 변수 func에 할당하는 부분입니다.
함수 내용은 중괄호 {} 안에 위치하며, 여기서는 "02. 함수가 실행되었습니다"라는 문자열을 콘솔에 출력하고 있습니다.

func();: 이 부분은 변수 func에 할당된 함수를 호출하는 부분입니다.
변수에 할당된 함수를 호출하면 함수 내부의 코드가 실행되고, "02. 함수가 실행되었습니다"가 콘솔에 출력됩니다.

결과 확인하기
02. 함수가 실행되었습니다
